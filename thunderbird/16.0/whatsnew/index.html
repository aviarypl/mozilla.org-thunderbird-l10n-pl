Silent, Background Updates

Thunderbird will now download and apply updates in the background allowing you to start quickly the next time Thunderbird starts up. This eliminates the update progress dialog on all platfroms and on Windows, it removes the need for the User Access Control dialog, making updating easier and quicker.
