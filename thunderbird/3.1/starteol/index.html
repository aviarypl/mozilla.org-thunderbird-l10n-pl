It's Time for a Change!

We want to thank you for your loyalty to the Mozilla project and for using Thunderbird, our email software. This is awesome!

Sometimes it is time for a change. Starting on April 24, 2012, we will stop supporting Thunderbird version 3.1.x. What this means is, while you can continue to use Thunderbird 3.1, we will not be making improvements to Thunderbird 3.1 which would include security and performance fixes.

Upgrade to the Latest Version of Thunderbird Now

By upgrading to the latest version of Thunderbird now, you will:

  • work in a safer environment, you'll benefit from all the latest security fixes and improvements
  • experience better performance because we continually work to improve responsiveness on a regular basis
  • and enjoy many new features that are already appreciated by millions of users.

You will find more details about the latest new features here.

Do you want to start now? Go to the Help menu in Thunderbird and select 'Check for Updates'. If it doesn't work you can visit getthunderbird.com.

Need Upgrade Assistance?

If you don't feel comfortable upgrading or if you have questions after you have upgraded, please check out our Thunderbird 3.1 End of Life FAQ article.

If that doesn't answer your question, please file a support request to our community of support volunteers with what you did, what happened, what you expected, your Thunderbird add-ons if any, your anti-virus software if any, mail provider and ISP and any other troubleshooting details.

More Ways to Help Us

Get Involved! You don't have to be technical to help us and being involved is good for people wanting to get more familiar with Thunderbird. You can contribute at many levels: testing, localization, technical support, and suggesting new features.
